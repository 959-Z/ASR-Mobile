# License Notes

This project scaffold does not include model weights, sample audio, or third-party native source code.

Before final submission, document licenses for every external component actually used.

## Components to check

| Component | License/status | Notes |
|---|---|---|
| Android project code in this folder | Team/project code | Choose a license if publishing publicly |
| whisper.cpp source | Check upstream license | Only include after approval/manual placement |
| Whisper model weights | Check model license and usage terms | Do not commit large model binaries by default |
| Sample audio | Must be owned, generated, or openly licensed | Record your own samples if possible |
| Desktop baseline script | Course repository material | Reference only unless copied/adapted |

## Recommended practice

- Record your own short audio samples for English, French, and Chinese tests.
- Keep model files outside Git.
- If publishing to GitHub, include clear attribution and license information.
